# Lumo

**Building modern, open-source software for a faster, more private web.**

Lumo is an open-source organization focused on creating software that is fast, secure, and privacy-first. Our flagship project, **Lumo Browser**, is designed to provide a modern browsing experience while remaining lightweight, transparent, and community-driven.

## Our Projects

* **Lumo Browser** — A modern, privacy-focused web browser.
* **Lumo Core** — Shared libraries and core components.
* **Lumo Tools** — Utilities for development and automation.
* **Lumo Docs** — Documentation and developer resources.

## Contributing

We welcome contributions from developers, designers, security researchers, and technical writers. Check the `CONTRIBUTING.md` file in each repository to get started.
